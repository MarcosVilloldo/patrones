## Instalación

1. Instala las dependencias:
```bash
npm install
```

## Uso

2. Inicia el servidor de desarrollo:
```bash
npm run dev
```

3. Abre tu navegador en la URL que aparece en la terminal (`http://localhost:5173`)

## Scripts disponibles

- `npm run dev` - Inicia el servidor de desarrollo
